🆘 HÄUFIGE PROBLEME:

❌ "Visual Studio not found"
   → Installiere Visual Studio mit C++ Workload
   → Siehe: HARDWARE_SETUP.md → Troubleshooting

❌ "gPhoto2 not found"
   → Installiere gPhoto2 von https://sourceforge.net/projects/gphoto2/
   → Oder: winget install gPhoto2.gPhoto2

❌ "Camera not recognized"
   → Überprüfe USB-Verbindung
   → Teste mit gPhoto2 --list-cameras
   → Siehe: HARDWARE_SETUP.md → Nikon Setup

❌ "Printer not found"
   → Registriere den Drucker als Windows-Drucker
   → Siehe: HARDWARE_SETUP.md → Canon Setup

❌ "App startet nicht"
   → Führe aus: flutter clean && flutter pub get
   → Starte neu: flutter run -d windows
   → Bei Fehler: Lese Logs mit: flutter run -v
